# v3.11.0 - 2026-01-02

## 저장소 구조 개편

공식 Claude 플러그인 구조 채택 및 저장소 단순화.

### Changed

- **저장소 구조 대폭 개편**: 공식 플러그인 구조 채택
  - `plugins/` 폴더 도입: 모든 플러그인을 `plugins/` 하위로 이동
  - `workflow-bundle/skills/` 폴더: 스킬들을 `skills/` 폴더로 이동 (자동 인식)
  - MCP 설정 분리: marketplace.json의 `mcpServers` → 각 플러그인 `.mcp.json`으로 분리
  - 플러그인 메타데이터 추가: 각 플러그인에 `.claude-plugin/plugin.json` 추가

- **marketplace.json 간소화**: 카탈로그 역할만 수행
  - `mcpServers`, `skills`, `agents` 제거
  - `category`, `homepage`, `tags` 추가

### Removed

- **docs/ 폴더 제거**: ADR은 Serena 메모리로 대체
- **.archive/ 폴더 제거**: 불필요한 아카이브 정리

### Technical Details

**새로운 저장소 구조**:
```
wogus-plugin/
├── .claude-plugin/marketplace.json  # 카탈로그
├── plugins/
│   ├── workflow-bundle/
│   │   ├── .claude-plugin/plugin.json
│   │   ├── .mcp.json
│   │   ├── skills/
│   │   │   ├── analyze/
│   │   │   ├── plan/
│   │   │   ├── execute/
│   │   │   ├── record/
│   │   │   └── mr-review/
│   │   └── agents/
│   ├── terraform/
│   ├── amplitude/
│   ├── slack/
│   └── atlassian/
├── CLAUDE.md
├── README.md
└── CHANGELOG.md
```

**각 플러그인 통일 구조**:
```
{plugin}/
├── .claude-plugin/plugin.json  # 메타데이터
├── .mcp.json                   # MCP 서버 설정
└── README.md                   # 문서
```
